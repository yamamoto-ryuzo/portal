# ゆかりの1日分具体メニュー（ガム2本・おやつ5粒を維持）

前提
- 体重: 5.6 kg、BCS: 6
- フード: 朝35 g / 夜40 g（合計75 g）
- 歯磨きガム: 2本/日（合計17.2 kcal）
- おやつ: 5粒/日（合計10.5 kcal）
- フードのエネルギー: 260 kcal/100 g（2.6 kcal/g）
- 便は良好で、これ以上食事量を増やすと柔らかくなるため量は維持する方針

目標
- BCS: 5（体重目安 約5.0 kg）
- 歯磨きガム: 2本/日（合計17.2 kcal）維持
- おやつ: 5粒/日（合計10.5 kcal）維持
- フードのエネルギー: 260 kcal/100 g（2.6 kcal/g）
- 期間目安: 8〜12週間（安全な減量率: 週0.5〜1%）

## 方針

**「量は減らさず、消費を増やし、質で補う」**

| ステップ | 内容 | タイミング |
|----------|------|------------|
| 1 | 計測・記録を正確にし、隠れカロリーをゼロにする | 今すぐ（最優先） |
| 2 | 運動量を増やす（毎日45分・インターバル散歩） | 1〜4週目 |
| 3 | フードを低脂肪・高タンパク・高繊維の減量用に切替 | 4週間後も効果なしなら |
| 4 | 獣医で代謝・ホルモン検査 | 8週間後も効果なしなら |

### 方針の根拠

**① フードをさらに減らしてはいけない理由**
- 現在の摂取カロリー（フード195 kcal + ガム17.2 kcal + おやつ10.5 kcal ≒ 223 kcal/日）は、
  5.6 kgのポメラニアンの推定維持量（RER×1.4〜1.6 ≒ 357〜408 kcal/日）を大きく下回っている。
- これ以上減らすと、筋肉量の減少・基礎代謝の低下・低血糖リスクを招き、長期的に体重管理がさらに困難になる。

**② 運動を先に増やす理由**
- 消費カロリーを増やすことで、食事量を変えずに脂肪を減らしBCSを改善できる。
- 散歩や室内遊びを増やしても便への影響は起きない（食事の変更と異なり消化器への負担がない）。
- 精神的満足も高まり、要求行動（おねだり）が減り、結果的に総摂取カロリーの管理がしやすくなる。

**③ 効果がなければフードの「質」を変える理由**
- 同じ量（75 g）のまま低脂肪・高繊維・高タンパクの減量用フードに切り替えることで、
  カロリー密度を下げながら満腹感・栄養バランスを維持できる。
- 量を変えないため便への影響を最小化しながら減量効果が期待できる。

## 注意: フード量削減は原則行わない
現状の摂取カロリー（フード75 g + ガム + おやつ ≒ 223 kcal/日）は、
推定必要維持量（RER×1.4〜1.6 ≒ 357〜408 kcal/日）より既に大幅に少ない。
さらなる量の削減は筋肉量減少・基礎代謝低下・低血糖リスクがあり推奨しない。

## BCS改善のために行うこと

### 1. データ確実化（最優先・今すぐ開始）
- キッチンスケールでフードを毎回g計量（目分量禁止）
- 体重は週1回・朝・排尿後の同条件で記録
- 2週間の完全ログを開始（フードg・ガム本数・おやつ粒数・散歩時間・便スコア）
- 家族全員が「与えたものは全て記録」ルールを守る

### 2. 隠れカロリー排除
- 家族からの"ちょいあげ"をゼロにする（記録に残らないカロリーが肥満の主原因になりやすい）
- トッパーや調理食品を与えている場合は一時中止してログで確認

### 3. 行動介入（給餌法の改善）
- パズルフィーダーに朝または夕のフードの一部を入れて給餌時間を延ばす
  - カロリーは"追加"ではなく日々のフード総量（75 g）の中から入れる
  - 初期は簡単な難易度から始め、慣れたら上げる
- 朝夕の食事を小分け（3〜4回）にして満腹感を維持
- 昼に嗅覚遊び（おやつを隠す・探す）を5〜10分導入

### 4. 運動増加（段階的に）
- 目標: 毎日合計45分（朝20分 + 夕方25分）
- 散歩にインターバルを取り入れる（速歩2〜3分 → ゆっくり1分の繰り返し）
- 帰宅後に室内での駆け回り遊びを1〜2回追加

### 5. フード組成の変更（4週間で効果不十分なら）
- 同じ量（75 g）で低脂肪・高繊維・高タンパクの減量用フードへ切替を検討
- 切替は1週間かけて徐々に行い、便の状態を毎日観察する

### 6. 獣医相談（以下の場合は早めに）
- 4〜8週間ログを続けてもBCS・体重が改善しない
- 元気低下・ふらつき・食欲不振・便の著しい悪化が起きた場合
- 甲状腺や代謝疾患の可能性を血液検査で除外する
