# SSOT + LoRA + RAG 統合アーキテクチャ（草案）

概要: ハルシネーションを極小化し実業務で本当に動くLLMシステムを作るための、SSOT(信頼できる唯一の情報源)・LoRA・RAGの統合アーキテクチャ設計・実装ガイドです。Dify・LangGraph・Copilot Studioを対象に、オーケストレーター〜回答統合までの8エージェント構成を段階的に構築します。
