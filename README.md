# portal

このリポジトリは静的サイト（Jekyll）ベースのドキュメントを含んでいます。GitHub Pages で公開しています。

- サイト（Pages）：https://yamamoto-ryuzo.github.io/portal/

使い方
- リポジトリのルートにある `index.md` がサイトのトップです。
- ローカルで確認するには Jekyll 環境を用意してビルドしてください（例: `bundle exec jekyll serve`）。

貢献
- プルリクエスト、イシュー歓迎です。変更は Markdown ファイルとして追加してください。

ライセンス
- リポジトリに記載のあるライセンスに従ってください。
